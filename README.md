# CFE_for_C
A compiler, formatter and executer for C

## Environments
To use this repository, you need 
 * gcc 4.6.1
 * flex 2.5.4
 * bison 2.4.1

Run the following commands to test if gcc, flex and bison are installed
```
gcc -v
flex -V
bison -V
```
You should see the following result
